# Learn React – My React Learning Journey 🚀

This repository documents my journey of learning **React.js** through hands-on projects.
Instead of only studying theory, I built multiple small applications to understand core React concepts such as **components, hooks, state management, context API, and Redux**.

Each project in this repository focuses on a specific concept or feature of React. Together, they represent my progress from **basic React fundamentals to more structured and scalable applications.**

The goal of this repository is to **practice, experiment, and strengthen my frontend development skills using React.**

---

# Projects Overview

### Background Changer

A simple React application that allows users to dynamically change the background color of the webpage using buttons. This project helped in understanding **React components and the useState hook.**

---

### Password Generator

A tool that generates secure random passwords with customizable options. It demonstrates **state handling, user input control, and dynamic UI updates.**

---

### Currency Converter

A React application that converts currencies using real-time exchange rate data. This project focuses on **API integration, custom hooks, and component-based design.**

---

### Context API Demo

This project demonstrates how **React Context API** can be used to share data between components without prop drilling. It includes simple components like login and profile to show global state usage.

---

### Theme Switcher

An application that allows users to toggle between different UI themes (such as light and dark mode). This project explores **context-based theme management and reusable components.**

---

### Todo Application

A basic task management application where users can add, delete, and manage tasks. It helped practice **state management, component communication, and list rendering.**

---

### Redux Todo

An improved version of the Todo application built using **Redux for state management**. This project focuses on understanding **Redux store, actions, reducers, and global state handling.**

---

### Blog Project

A larger React project that includes blog creation and authentication features. It integrates services like **Appwrite for backend functionality**, demonstrating how React works with external services.

---

### React Practice App

A small experimental project used for practicing **React component structure, routing, and layout design.** It contains pages like About and Contact to simulate a basic website structure.
